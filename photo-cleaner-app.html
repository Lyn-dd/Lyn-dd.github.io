PHOTO CLEANER

이미지를 브라우저에서 분류한 뒤, 삭제 예정 목록을 만들고 Windows 휴지통으로 옮길 수 있게 도와줍니다.

이미지 폴더나 이미지 파일들을 여기에 드래그 앤 드롭하세요. 드래그가 안 되면 아래의 기본 파일 선택 칸을 사용하세요.

파일은 서버로 전송되지 않고, 브라우저 안에서 미리보기용으로만 읽힙니다.

JavaScript 초기화 중입니다. 이 문구가 계속 보이면 페이지 파일이 코드가 아닌 텍스트로 올라간 상태이거나, 브라우저가 스크립트를 막은 상태입니다.

준비 중입니다.
유지
삭제
미리보기를 표시할 수 없는 이미지 형식입니다. 분류는 계속 가능합니다.

마우스/터치로 왼쪽 스와이프하면 유지, 오른쪽 스와이프하면 삭제 예정입니다. 키보드는 ← 유지, → 삭제, Z 또는 Backspace 되돌리기입니다.

분류 완료

권장 처리 방식
1) 삭제 목록 JSON을 다운로드합니다.
2) PowerShell 처리 명령을 복사합니다.
3) Windows PowerShell 또는 Windows Terminal을 열고 붙여넣습니다.
4) 터미널 안내에 따라 JSON 파일 경로와 원본 이미지 폴더 경로를 넣으면 삭제 예정 파일만 휴지통으로 이동합니다.
